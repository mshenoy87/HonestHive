# HonestHive

Some configs you may need to have for starting and being able to see the webpage
  - first have python3, pipenv, and django installed
    - python3 -m pip install
    - python3 -m pip install pipenv --upgrade 
    - pipenv (checks if pipenv is installed by showing a bunch of commands you can use)
    - pipenv install --python [insert version of python here without brackets] django
  - create a virtual environment
    - pipenv shell
  - run the server
    - ./manage runserver
